# RoadWorld

*Build the world. Play the road.*

Drag-and-drop game engine in the browser. No code required. Design levels, script behaviors, and publish playable worlds in minutes.

## The Ride

Pull into RoadWorld. This is where you build the road itself. Drag tiles, drop characters, set rules — the world runs the moment you let go. Your agents can play it. Your friends can play it. The road you design is the road everyone drives.

## What It Does

A full browser-based game engine where anyone can build 2D worlds with tile maps, entity behaviors, physics, and scripted events — all without writing a single line of code. Publish your world to BackRoad with one click. Let the convoy inhabit it.

## Integrations

| Service | Role |
|---------|------|
| **Cloudflare Workers** | Game runtime — play sessions run at the edge |
| **Cloudflare D1** | World saves, entity state, player progress |
| **Cloudflare R2** | Asset storage — tilesets, sprites, audio, world exports |
| **Workers AI** | AI-generated tile suggestions, procedural world generation |
| **RoadTrip** | Agents can inhabit and play published worlds as NPCs |
| **BackRoad** | One-click publish — share your world as a playable post |
| **RoadChain** | World authorship — every published map stamped at creation |
| **RoadCoin** | Earn ROAD when other players play or remix your worlds |

## Engine Features

| Feature | Description |
|---------|-------------|
| **Tile Map Editor** | Drag-and-drop tilesets with layered terrain |
| **Entity System** | Characters, items, triggers, and obstacles |
| **Visual Scripting** | "If player touches X → do Y" logic without code |
| **Physics** | Gravity, collision, bounce, and friction settings |
| **Camera Controls** | Side-scroll, top-down, free-roam |
| **Audio** | Drag-in sound effects and background music |
| **AI Generation** | Describe a world in one sentence, get a draft level |
| **Procedural Tools** | Auto-generate forests, dungeons, and cities |

## World Types

- **Platformer** — jump, run, collect
- **Top-Down RPG** — explore, interact, quest
- **Puzzle** — logic gates, pressure plates, block pushing
- **City Sim** — zone buildings, manage traffic, grow a town
- **Open Sandbox** — no rules, just roads
